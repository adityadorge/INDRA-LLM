# INDRA-LLM
## Intelligent Deep Representation Architecture

**Building a Large Language Model from Scratch**

INDRA-LLM is an open-source research project focused on designing, implementing, training, and deploying a Large Language Model (LLM) entirely from scratch. The goal is to understand every layer of modern language models by building each component independently, from data preprocessing and tokenization to transformer architecture, distributed training, inference, and evaluation.
